• CREATE
  • ALTER
  • DROP

CREATE

O comando CREATE é usado para criar novas estruturas no banco de dados, como tabelas, views ou índices. Exemplo: CREATE TABLE clientes (id INT, nome VARCHAR(100));

ALTER

O comando ALTER modifica a estrutura de uma tabela existente, como adicionar ou remover colunas. Exemplo: ALTER TABLE clientes ADD email VARCHAR(100);

DROP

O comando DROP exclui completamente uma estrutura do banco de dados, como uma tabela ou view. Exemplo: DROP TABLE clientes;
