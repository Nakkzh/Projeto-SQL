INSERT

O comando INSERT é usado para adicionar novos registros em uma tabela do banco de dados. Exemplo: INSERT INTO tabela (coluna1, coluna2) VALUES (valor1, valor2);

UPDATE

O comando UPDATE modifica dados existentes em uma tabela. Exemplo: UPDATE tabela SET coluna1 = valor1 WHERE condição;

DELETE

O comando DELETE remove registros de uma tabela com base em uma condição. Exemplo: DELETE FROM tabela WHERE condição;

SELECT

O comando SELECT é usado para consultar dados em uma tabela. Exemplo: SELECT coluna1, coluna2 FROM tabela WHERE condição;
