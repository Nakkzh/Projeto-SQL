COUNT

Função: Conta o número de linhas ou entradas não nulas.
COUNT: Conta TODAS as linhas da tabela, independentemente de terem NULL ou não. É o "quantos registros existem no total?".
COUNT(coluna): Conta apenas as linhas onde a coluna específica não é NULL. É o "quantos registros têm um valor preenchido aqui?".

SUM

Função: Soma todos os valores em uma coluna numérica.
Ignora completamente os valores NULL. Eles não são tratados como zero, são simplesmente excluídos do cálculo.
Se todos os valores forem NULL, o resultado é NULL.

AVG

Função: Calcula a média aritmética (soma dos valores / número de valores).
Ela também ignora NULL. A média é calculada apenas com os valores existentes.
A fórmula é SUM(Valor) / COUNT(Valor). Perceba que o divisor não é COUNT(*).

MIN

Função: Encontra o menor valor em um conjunto.
Funciona com diversos tipos de dados:

  • Números: O menor número.
  • Texto: O primeiro na ordem alfabética (A-Z).
  • Datas: A data mais antiga.

MAX

Função: Encontra o maior valor em um conjunto. Funciona com diversos tipos de dados:

  • Números: O maior número.
  • Texto: O último na ordem alfabética (A-Z).
  • Datas: A data mais recente.
